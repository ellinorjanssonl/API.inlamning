Register

  • GET/newuser - returnerar alla befintliga användare.

  • GET /newuser/:id - returnerar en user med angivet id eller status 404 om användaren saknas.

  • POST /register - registrerar en ny användare med hashat lösenord med accepterar JSON objekt på formatet {"name": "", "email": "", "username": "password": ""}

  • POST /login - loggar in en användare som registrerats.

  • PUT /users/:id - uppdaterar användaren med angivet id, accepterar JSON-objekt på formatet {"name": "", "password": "", "email": ""}.
